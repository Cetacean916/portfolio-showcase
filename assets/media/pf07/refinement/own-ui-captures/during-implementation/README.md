# 작업 중 캡처 수락본과 남은 큐

아래 수락 항목은 패키지 소유 런타임과 보호된 연결 관측에서 직접 생성하고 공개용으로 정제했습니다.

- `HOME-004`: 실패 → 재시도 → 복구 흐름
- `HOME-005`: 구현한 연동 구성 기반 구조도 또는 증거판
- `CASE-010`: 정상 주문 관리 이벤트
- `CASE-011`: 실패·재시도 상태
- `CASE-012`: 복구 완료 상태
- `CASE-013`: reconcile 전·후
- `CASE-014`: n8n 성공·실패·재시도 증거판
- `CASE-015`: HubSpot Deal·Contact 증거판
- `CASE-016`: Slack 복구 알림
- `GUIDE-006`: HubSpot·Slack 설정 마법사
- `GUIDE-007`: 실제 최종 Linux server package의 정상 기동 화면 — 제품 동결 후 공개본으로 확정
- `GUIDE-008`: 실제 최종 Linux package의 HTTPS 터널 활성 제어 화면 — 제품 동결 후 공개본으로 확정

각 수락 파일의 언어·상태·출처·변형·크기·SHA-256는 상위 `final-public-asset-ledger.json`에서 확인합니다. `GUIDE-007~008` 공개본은 실제 기능 관측이 끝난 뒤 `after-completion/`에만 두며, 보호된 원본·런타임 경로는 공개 세트에 포함하지 않습니다.
