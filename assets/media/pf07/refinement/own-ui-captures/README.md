# PF07 자체 화면 캡처 큐

이 폴더에는 PF07 구현 결과에서 직접 만든 화면·증거판·구조도의 공개용 편집본만 둡니다. AI 이미지, 외부 문서 스크린샷, 마스킹 전 원본은 넣지 않습니다.

- `during-implementation/`: 구현과 테스트 중 상태가 생길 때 확보할 자료
- `after-completion/`: 최종 UI·패키지·CI가 고정된 뒤 확보할 자료

지금 사용자가 미리 찍을 자료는 없습니다. 작업자는 더미 주문과 더미 고객으로 필요한 상태를 만들고 즉시 원본을 확보합니다. 로그인·2FA가 필요할 때만 사용자에게 해당 인증을 요청합니다.

공개 파일은 토큰, webhook URL, 이메일, 고객명, 워크스페이스 ID, 로컬 사용자명과 절대경로를 제거하고 시각 검토를 마친 뒤 `PUBLIC-ASSET-MANIFEST.txt`에 파일 단위로 추가합니다.
